## 🎯 适用场景

本指南适用于以下场景：
- 新购买的 Mac 或 Windows 电脑

## 📝 注意事项

1. **备份重要数据**：配置前请备份重要文件
2. **网络环境**：部分工具需要稳定的网络连接
3. **权限管理**：某些配置需要管理员权限
4. **版本兼容**：注意工具版本兼容性
