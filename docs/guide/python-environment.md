# Python 环境配置

## 模块一览

| 模块 | 内容 |
|------|------|
| **[uv 与项目](./python-uv.md)** | 安装 uv、国内镜像、Python 版本、`uv init` / `venv` / `add` / `sync` |
| **[Notebook](./python-notebook.md)** | JupyterLab、Jupyter Notebook、`ipykernel` 注册内核 |
| **[工具与调试](./python-tooling.md)** | 开发依赖、`.env`、VS Code、`uv tool`、验证命令 |
